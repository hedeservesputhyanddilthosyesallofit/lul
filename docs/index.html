This is a JavaScript library for transliterating English text to the Shavian (Shaw) alphabet.

𐑞𐑦𐑕 𐑦𐑟 𐑱 ·𐑡𐑭𐑝𐑩𐑕𐑒𐑮𐑦𐑐𐑑 𐑤𐑲𐑚𐑮𐑧𐑮𐑰 𐑓𐑸 𐑑𐑮𐑨𐑯𐑟𐑤𐑦𐑑𐑩𐑮𐑱𐑑𐑦𐑙 ·𐑦𐑙𐑜𐑤𐑦𐑖 𐑑𐑧𐑒𐑕𐑑 𐑑 𐑞 ·𐑖𐑱𐑝𐑰𐑩𐑯 (·𐑖𐑭) 𐑨𐑤𐑓𐑩𐑚𐑧𐑑.

Try it

𐑑𐑮𐑲 𐑦𐑑

Usage

$ npm install to-shavian
const toShavian = require('to-shavian')
toShavian('Hello!')
//-> "𐑣𐑩𐑤𐑴!"
